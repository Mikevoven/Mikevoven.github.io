点到直线的距离公式

Posted on 2025-03-18

问题描述

给定直线 \(l:ax+by+c=0\) 和 \(A(x_1,y_1)\) ，求点 \(A\) 到直线 \(l\) 的距离.

解：记直线的法向量 \(w=(a,b)\) ，点 \(A\) 沿着直线的法向量走到点 \(B\) 上，移动的长度就是我们要求的距离，记作 \(λw\) ，如图所示

\(A\) 点跑了 \(λw\) 距离后到达 \(B\) 点，此时 \(B\) 的坐标为 \(A+\lambda w\) ，带入直线方程得 \[ w\cdot (A+\lambda w) + c = 0 \] 解得 \[ d = |\lambda w| = \frac{|w\cdot A + c|}{|w|} = \frac{|ax_1 + by_1 + c|}{\sqrt{a^2+b^2}} \]
